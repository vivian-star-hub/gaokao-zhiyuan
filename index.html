等位分换算

输入基准年分数，自动换算成其他年份相同位次对应的"等效分数"。

例如："我今年考了 580 分，相当于去年多少分？"

冲稳保推荐（位次法）

输入分数和科类，系统按位次在三个档位筛选院校。位次越小越靠前，门槛越高。

（位次偏移：冲=高排名、稳=同位次附近、保=低排名）

选科匹配（3+3 模式）

输入你的 3 门选考科目（首选 1 门 + 再选 2 门），筛选可报考的院校专业组。

多年度趋势对比

输入院校名称与科类，查看该校 2021-2025 五年的投档线/位次走势（轻量 SVG 折线图，无外部依赖）。

本地数据导入 / 解析工具

用于以本地 .xls / .xlsx 文件生成可上传的 JSON（运行完全在浏览器内，不上传任何文件）。

📖 使用说明

  1. 从 sdzk.cn 下载原始 .xls 文件（无需手动预处理）
  2. 选择类型（默认“自动识别”）和年份（自动推断）
  3. 点击下方“下载转换后的 JSON”
  4. 将生成的 JSON 覆盖到 data/ 目录，重载页面

⚠️ 作用：本工具让“非技术用户”也能更新本项目的数据，避免动手写代码。本项目本身提供 import.py（Python 版本，适用于本地批量处理）。

关于本项目

本项目为山东新高考（3+3 模式）志愿填报指导手册的网页版。

数据来源：山东省教育考试院 sdzk.cn

使用须知：所有数据需对照官方公告逐条核验后再用于实际填报。

⚠️ 数据为占位骨架，正式使用前请从山东省教育考试院逐年核验
