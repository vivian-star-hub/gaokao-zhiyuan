# 山东高考志愿填报指导手册

山东新高考（3+3 模式）志愿填报指导手册的网页版。

## 项目结构

```

├── css/

├── js/
│   ├── data.js                 # 核心数据 + 算法（5 年真实数据 + 函数 + 5 年 lazy-load）
│   └── app.js                  # 应用逻辑（查询 / 展示 / Tab 切换 / SVG 折线图）
├── data/
│   ├── toudang-index.json      # 投档表索引

│   ├── toudang-2022.json       # 2022 年投档表（~3.8MB）

    ├── 2021-2025 一分一段表 xls (5 个)

    ├── 2021-2021 各轮次投档表 xls (5 个)
    ├── 2024-pcx.png            # 2024 批次线图片（OCR 提取）
    ├── 2024-pcx.txt            # 2024 批次线 OCR 文本
```

抓取方法：

- 数据解析：xlrd (.xls) + pdfplumber (PDF) + tesseract OCR (图片)
- 物理/历史分类启发式：基于专业名称 + 院校名称关键词

  - 不确定项默认归物理（占招生大头的多数）

## 8 个功能 Tab

1. **位次查询** — 分数 ↔ 位次互查
2. **等位分** — 跨年分数换算
3. **冲稳保** — 按位次分档推荐院校（5 年任选，按需 lazy-load）
4. **选科匹配** — 按 3+3 选科筛院校专业组（演示数据 30 条）
5. **趋势对比** — 单校 5 年位次折线图（自动 lazy-load 5 年）
6. **批次线** — 各批次控制线
7. **一分一段** — 完整一分一段表
8. **数据状态** — 核验状态总览（5 种徽章）

- 趋势对比"全部 5 年"时并行加载

## 已知局限

- **物理/历史分类启发式**：sdzk.cn 投档表不直接标"首选物理/历史"，按专业名称 + 院校名称启发式分类。
  - 物理:历史 ≈ 3.4:1（优化后，原 5:1）
  - 不确定项归物理（多数招生在物理）
  - 修正方向：可结合"专业组选科要求"进一步判断
- **投档表不含"最低分"**：sdzk.cn 自 2021 起就只公布"最低位次"，不公布"最低分"——这反映了山东的"位次改革"
- **选科匹配的招生数据为"剩余计划"**：我们从 sdzk.cn 抓取的 6,374 条是退档后剩余计划，主要覆盖专科批选科要求 + 学费信息。如需完整 30 万+ 条原始计划需走 PDF 解析。
- **艺术/体育类已导入（15,433 条）** ✅：体育 / 美术与设计 / 音乐 / 舞蹈 / 播音 / 书法 / 表(导)演 7 类齐全，按综合分存储。
- **专科批投档表**：5 年全部已合并到本科批投档表中（去重时取最小位次），但实际上专科批位次逻辑不同——目前混在一起使用

## 数据导入工具

本项目提供三种数据导入方式：

### 1. 浏览器导入（推荐）🆕
打开页面 → "数据导入" Tab → 拖入 .xls/.xlsx 文件 → 自动识别类型 → 下载 JSON。

实现：`js/importer.js` + SheetJS (xlsx.js) CDN
**完全本地运行，文件不上传任何服务器。**

支持类型：
- 一分一段表
- 普通类投档表（物理/历史合并）
- 艺术/体育类投档表（按类别自动识别）
- 院校专业招生计划（含选科要求 + 学费）

### 2. Python 命令行脚本
```bash
# 把 .xls 放到 data-raw/ 目录，文件名建议含 YYYY + 类型关键词
python3 import.py                       # 处理 data-raw/ 所有文件
python3 import.py --file path/to/x.xls  # 处理单个文件
python3 import.py --year 2025 --dry-run # 试运行不写文件
```

### 3. 直接调用 sdzk.cn
参考 `/tmp/scan_*.py` 和 `/tmp/dl_*.py` 脚本：
- `scan_news.py` 扫描 NewsID 列表
- `download_toudang.py` 下载 + 解析投档表
- `download_toudang_art.py` 下载 + 解析艺体类
